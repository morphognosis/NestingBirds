Nesting birds running the Mona neural network.

To build:

UNIX:
Execute 'make' to make executables.

Windows:
Start Microsoft Visual Studio using solution.

Directories:

1. src:
- mona: mona neural network.
- common files.
- nestingbirds: nesting birds.
2. bin: executables.

Commands:

nestingbirds: run nest-building birds program.
